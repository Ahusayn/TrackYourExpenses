
Expense Tracker App

A simple app to track and visualize your expenses. It displays transactions by month and includes a daily expense chart using SwiftUI Charts. The app fetches mock transaction data from JsonBin.

Features

Transaction List: View transactions grouped by month.
Daily Expense Chart: Visualize daily spending trends with a bar chart.
Categories: Organize expenses by custom categories.
Tech Stack

-SwiftUI for UI
-SwiftUI Charts for visualizations
=JsonBin for mock API data
